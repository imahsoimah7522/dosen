## 🌟 Tentang Dosenova

**Dosenova** adalah platform CMS (Content Management System) berbasis web yang dirancang khusus untuk dosen dan akademisi. Platform ini memungkinkan pengguna untuk membangun dan mengelola website portfolio profesional secara mudah tanpa perlu keahlian pemrograman.

### Fitur Utama

- 🏠 **Website Portfolio Publik** — Tampilan modern dan responsif untuk menampilkan profil akademik
- 👤 **Manajemen Profil** — Edit biodata, foto, motto, dan informasi pribadi
- 💼 **Pengalaman Kerja** — Catat riwayat karir dan pengalaman profesional
- 🎓 **Mata Kuliah** — Kelola daftar mata kuliah yang diampu
- 📄 **Penelitian & Publikasi** — Dokumentasikan jurnal, paper, dan karya ilmiah
- 🚀 **Produk & Hak Kekayaan Intelektual** — Showcase software, dataset, dan inovasi
- 🤝 **Pengabdian Masyarakat** — Catat kegiatan pengabdian kepada masyarakat
- 📬 **Pesan Kontak** — Terima pesan dari pengunjung website
- 🔐 **Admin Dashboard** — Panel administrasi lengkap dengan autentikasi aman
- 🎨 **Tema & Layout** — Pilih tampilan tema dan layout sesuai preferensi
- 🔑 **Lupa Password** — Reset password melalui email yang terdaftar

## 📊 Tech Stack

| Layer | Teknologi |
|---|---|
| Frontend | HTML5, CSS3, Vanilla JavaScript (ES6+) |
| Animasi | AOS (Animate on Scroll), GSAP |
| Backend | Supabase (PostgreSQL) |
| Autentikasi | Supabase Auth (Email/Password) |
| Deployment | Netlify / Vercel / GitHub Pages / cPanel |

## 🚀 Quick Start

1. Clone atau download repository ini
2. Jalankan server lokal:
   ```bash
   node server.js
   ```
3. Buka `http://127.0.0.1:8080/install.html` untuk memulai instalasi
4. Ikuti panduan instalasi yang tersedia di halaman installer

> 💡 **Catatan:** Pastikan Anda sudah memiliki akun [Supabase](https://supabase.com) untuk menggunakan fitur backend.
